# CJC-CLINIC

Minimal clinic management system.

Quick start

1. Initialize git (done locally by the assistant).
2. Create a GitHub repository and add it as `origin`.
3. Push the `main` branch: `git push -u origin main`.

Files of interest

- `public/` — PHP entry points and API.
- `src/` — frontend TypeScript and CSS.
- `config/` — PHP configuration.
